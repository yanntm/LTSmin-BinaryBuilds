This page hosts content regularly rebuilt using the continuous integration service Travis. We are particularly grateful to these companies for supporting quality open source by providing us these services for free.

This page hosts binaries used in our tool chains, and built from LTSmin, from the FMT group based in Twente, NL :

  • LTSmin (see LTSmin homepage)
These artifacts are not very portable, they're used in our build system, so are specific to the versions (gcc, glibc...) that live on travis-ci (Linux, OSX targets) or AppVeyor (Windows target). The build is configured to only build what we depend upon, we don't aim to build all of LTsmin in this build.

This software is a free open source package, visit LTSmin homepage for license information.

Please see the sources of this project to see exactly how we build them on Travis and AppVeyor.
